## Customer

A **Customer** can be registered through the mobile app or by a branch office (ERP user) in the system.While Creating Customer Name and Mobile number are mandatory intially,

### Customer Information

Customers (users) can create and update the following details:

#### 1. Personal Information
- Name  
- Mobile Number  
- ID Number  
- Nationality  
- Gender  
- Date of Birth (DOB)  
- Female Contact Person Number (required if the labor is a female candidate)  

#### 2. Work Information

#### 3. Location Information

#### 4. Bank Details

#### 5. Family Information

---

### Customer Delegation (to contact assistant of the customer )

Customers can assign a contact person for communication.
Customer can add multiple delegation.

**Required Details:**
- Name  
- ID Number  
- Mobile Number  
- Status (Active / Inactive)  

**Activation:**
- The delegation will be activated only after verification through the customer's registered mobile number.
