# Employee Inquiry

  - Used to filter and search employee profile details and identity information.

  - Helps quickly locate specific employees based on key identifiers and profile attributes.

### Overview:

  - Allows users to search and inquire employee information using the following fields:

    - Name
    - Employee ID
    - Iqama Number
    - Country Entrance Number
    - Mobile Number
    - Company Reference Number
