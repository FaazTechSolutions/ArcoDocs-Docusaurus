# Lodging Profile Search

  - This page is just a view purpose only, no actions and workflows are there.

  - This menu purpose is to filter the employee profiles.

  - Employee Profile search may filter by,

    - Employee Id
    - Employee Type
    - Gender
    - Employee Status
    - Lodge Id
    - Job Specification
    - Nationality
    - Age
    - Visa Profession
    - Experience
